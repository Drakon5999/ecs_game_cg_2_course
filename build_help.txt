На Windows:
Если есть visual studio, то просто открыть проект из vcproj
и собрать Release/x64 (Debug/x64).
Если нет, то самим собрать из исходников. Зависимости в deps/win.
Выходной каталог: build/win (build/win/debug)

На Linux:
Собрать через cmake. По идее ничего устанавливать не нужно.
Если не соберется, попробуйте sudo apt-get install libsfml-dev
(подтянет зависимости).
cmake рассчитан на систему с X11 Window System.
Выходной каталог: build/linux

На macOS:
Разбирайтесь сами)
Наверное, через cmake должно нормально собраться.

Для запуска скинуть содержимое папки assets в папку с бинарником.
Для windows еще нужно докинуть .dll из deps/win в эту же папку.
